### 4. Coding: Python Function for Prime Numbers

This solution uses a helper function `is_prime` for clarity and efficiency. The main function then iterates through the list and uses the helper to build the final list of primes.

```python
def get_primes(number_list):
    """
    Takes a list of numbers and returns a new list containing only the prime numbers.

    Args:
        number_list: A list of integers.

    Returns:
        A list of integers containing only the prime numbers from the input list.
    """
    
    def is_prime(n):
        """Helper function to check if a single number is prime."""
        if n < 2:
            return False
        # Check for divisors from 2 up to the square root of n
        for i in range(2, int(n**0.5) + 1):
            if n % i == 0:
                return False
        return True

    prime_numbers = []
    for num in number_list:
        if is_prime(num):
            prime_numbers.append(num)
            
    return prime_numbers

# Example usage:
```

### 5. Common Sense / Context Understanding: Ice in Coffee

**What happens:** The ice melts, and the coffee gets colder.

**Why:** This is due to the Second Law of Thermodynamics. Heat energy naturally flows from a hotter object to a colder one until they reach a state of thermal equilibrium (the same temperature).

1.  The hot coffee has a high amount of thermal energy (its molecules are moving rapidly).
2.  The ice has very little thermal energy (its molecules are locked in a crystal structure and vibrating slowly).
3.  When the ice is placed in the coffee, the heat from the coffee transfers to the ice.
4.  This energy transfer causes two things: the coffee's molecules slow down, making the coffee cooler, and the ice's molecules vibrate faster, breaking their solid bonds and causing the ice to melt into liquid water. This process continues until all the ice has melted and the entire liquid reaches a single, cooler temperature.

### 6. Math & Estimation: Piano Tuners in NYC (A Fermi Problem)

2.  **Number of Households:** Assume an average of 2.5 people per household.

3.  **Fraction of Households with Pianos:** Pianos are not extremely common. Let's estimate that 1 in 100 households owns a piano.
    *   3,200,000 households / 100 ≈ 32,000 pianos in homes.
4.  **Institutional Pianos:** Add pianos in schools, concert halls, churches, bars, etc. Let's add about 10% to our home estimate.
    *   32,000 * 0.10 ≈ 3,200 institutional pianos.
    *   Total Pianos: 32,000 + 3,200 ≈ 35,000 pianos.
5.  **Tuning Frequency:** Assume the average piano is tuned once per year.
    *   Total Tunings per Year: 35,000 tunings.
6.  **Workload of One Tuner:** How many pianos can one tuner service in a year?
    *   Assume they work 5 days a week for 50 weeks a year: 5 * 50 = 250 workdays.
    *   A tuning takes a couple of hours, plus travel time. Let's say a tuner can average 2 tunings per day.
    *   Tunings per Tuner per Year: 250 days * 2 tunings/day = 500 tunings.
7.  **Final Calculation:**
    *   Number of Tuners = (Total Tunings Needed) / (Tunings per Tuner)
    *   35,000 / 500 = **70 tuners**.
